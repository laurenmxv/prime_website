Executive Summary & Goals

Current Site — What's Broken

The current primeoutdoorexperts.com is a 6-page brochure site. It has no blog, no location pages, no individual service pages, no schema markup, and no property-type commercial pages. The competitor research across 3 keyword sets reveals that the site is invisible for most high-value commercial landscaping searches in the Orlando market.

Critical Gap

No Individual Service Pages

All 12 services compressed into one page. Zero keyword targeting per service.

Critical Gap

No Location Pages

5 counties served, zero suburb-specific pages. Losing all "[service] [city]" searches.

Critical Gap

No Blog / Content

Blog 404s. Zero authority-building content. No topical clusters.

Critical Gap

No Commercial pSEO

No property-type pages (HOA, office park, retail, medical). Zero competitors have built these — first-mover window is open NOW.

SEO Issue

No Meta Descriptions

Google writing its own snippets. CTR significantly below potential.

SEO Issue

No Schema Markup

No rich snippets, no FAQ schema, no LocalBusiness schema for map pack authority.

CRO Issue

Weak Social Proof

No star ratings, no review counts, no project photos with before/after context.

CRO Issue

Generic Hero Copy

"Orlando's Trusted Outdoor Experts" is indistinguishable from 15 competitors.

Prime OE can be the first company with a page for "HOA landscaping Winter Park FL," "office park landscaping Lake Nona," and 30+ other high-intent combos. Component Principles

Dark Sections

Hero, CTA bands, and feature highlights use #0A0A0A background with chartreuse accents. Creates premium, modern feel.

Light Sections

#FFFFFF or #F5F5F5 backgrounds for service lists, testimonials, FAQs. High readability for long-form content.

Chartreuse CTAs Only

Reserve #C8F000 exclusively for primary CTAs and key accent elements. Dilution kills conversion rate.

Photography Style

High-contrast landscape photography. Avoid stock. Real job photos with before/after pairs. Drone shots for commercial properties.

Icon Style

Outlined, single-weight icons (Lucide or Phosphor). Never filled icon sets alongside outlined — pick one.

Motion / Animation

Subtle scroll-triggered reveals. No auto-play video in hero. CLS score must stay under 0.1 — no layout-shifting animations.

Technical SEO Foundation

Core Web Vitals Targets

LCP Target

< 2.0s

Largest Contentful Paint. Hero image must be preloaded with fetchpriority="high". Use WebP/AVIF formats.

INP Target

< 150ms

Interaction to Next Paint. Minimize JS execution on main thread. Defer non-critical scripts.

CLS Target

< 0.05

Cumulative Layout Shift. Set explicit width/height on all images and embeds. Reserve space for lazy-loaded content.

TTFB Target

< 400ms

Time to First Byte. Use a CDN (Cloudflare). Server response under 200ms. Enable HTTP/3 where supported.

URL Canonical Rules

# Every page: self-referencing canonical tag
<link rel="canonical" href="https://primeoutdoorexperts.com/[current-path]/" />

AI SEO Requirements

🤖
~45% of Google searches now show AI Overviews. ChatGPT and Perplexity are growing fast. Content that earns citations in AI answers gets exposure even when the user never clicks. Structured, authoritative content is the path to AI citation.

Content Structure Rules for AI Extractability

Every service, commercial, and location page must follow this structure to maximize AI Overview and LLM citation rates:

ElementRequirementWhy It Works
Opening definitionFirst 50 words must contain a clear definition/description of what the page is about. "Prime Outdoor Experts provides HOA landscaping services in Winter Park, FL — including common area maintenance, entrance feature design, irrigation management, and seasonal color rotations."AI systems extract the first clear definition. This becomes the citation text.
Statistic blocksEach commercial page must include at least 2 cited statistics. e.g.: "HOA communities with professional landscaping see property values increase 15–20%, according to the National Association of Realtors (2023)."+37–40% AI citation boost from cited statistics (Princeton GEO study)
FAQ section4–6 natural-language Q&As at bottom of every service and commercial page. Questions should match how buyers actually search.FAQ schema enables rich snippets AND AI extracts Q&A pairs directly
Step-by-step sectionsUse numbered lists for "how it works" or process content. e.g. "How Our Commercial Landscaping Process Works: 1. Free property assessment… 2. Custom maintenance plan… 3. Scheduled crew visits…"AI systems prefer numbered process content for "how to" queries
Comparison tablesCommercial pages comparing property types or service tiers should use HTML tables, not bullet prose.Tables are extracted directly by AI systems for comparison queries
Author attributionBlog posts must show author name, title, and credentials. e.g.: "Written by [Name], Certified Landscape Professional (CLP), 15 years serving Orlando."+25–30% AI citation boost from expert attribution
Last-updated dateVisible "Last updated: [Month Year]" on all blog posts and guides.AI systems weight content recency heavily

Critical Developer Notes

Implement all 301 redirects on Day 1

Before any SEO work. Lost redirect chains = lost link equity and 404 errors for anyone with old URLs.

No lazy loading on above-fold images

Hero images must use fetchpriority="high" and NOT loading="lazy". This is a common mistake that tanks LCP scores.

Self-hosted Google Fonts

Do not load Google Fonts from fonts.googleapis.com. Download and host locally. External font loading adds 200–400ms TTFB penalty.

Trailing slash consistency

Pick one convention (trailing slash recommended) and implement server-level 301 redirect for the opposite. Do not allow both versions to return 200.

Schema validation before launch

Test all schema with Google Rich Results Test. Fix all errors before launch — broken schema is worse than no schema.

Mobile-first build

60%+ of landscaping search traffic is mobile. Build and test mobile first. The sticky bottom CTA bar is a mobile-specific element that must be built intentionally.
